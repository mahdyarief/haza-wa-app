# 🚀 HAZA-WA Desktop
**The Ultimate WhatsApp Gateway & Management Dashboard — Desktop Edition**

HAZA-WA is a professional, multi-session WhatsApp Gateway, Dashboard, and Automation System. This repository hosts the official desktop distributions (Windows, macOS, Linux) built with Tauri.

## 🌟 Key Features

### 📱 Core Messaging & Management
- **Multi-Session Management**: Connect and manage unlimited WhatsApp accounts simultaneously via QR code.
- **Pro WhatsApp Engine**: Powered by `@whiskeysockets/baileys` for high-speed, stable WebSocket connections.
- **Advanced Scheduler**: Plan messages with full media support (Images, Video, Docs).
- **Safe Broadcast**: Anti-ban mechanisms with randomized delays and batch processing.
- **Smart Auto-Reply**: Context-aware keywords matching for Groups, Private, or All.

### 🤖 Built-in AI Intelligence (The "HW" Logic)
- **Local AI Engine**: Integrated AI assistant that runs locally on your machine.
- **OCR (Image to Text)**: Automatically extract text from images sent in chats.
- **STT (Voice to Text)**: Transcribe voice notes and audio messages instantly using local models.
- **Stewardship**: AI understands context from images and audio as if it "saw" or "heard" them directly.

### 🛡️ Enterprise Grade Control
- **Webhook Integration**: Real-time event forwarding for messages, status changes, and group updates.
- **Open API (Swagger)**: Fully programmable RESTful API with over 100+ endpoints.
- **Granular Access**: Full Whitelist & Blacklist support for bot commands and replies.
- **Rich Contacts**: Management with profile pictures and verified business names.

### 🎨 Creative & Utility Tools
- **Sticker Maker**: Create stickers instantly with integrated background removal.
- **System Health**: Monitor server/application performance directly from the dashboard.
- **Top-Tier UX**: Modern "Arctic Frost" design with dark mode, glassmorphism, and smooth animations.
